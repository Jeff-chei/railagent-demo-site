# Fixed accessibility voice assets

The WAV files in this directory were generated for the RailAgent noncommercial
competition demo from the following Meta MMS-TTS checkpoints:

- Taiwanese: `facebook/mms-tts-nan`
- Hakka: `facebook/mms-tts-hak`
- Vietnamese: `facebook/mms-tts-vie`
- Thai: `facebook/mms-tts-tha`
